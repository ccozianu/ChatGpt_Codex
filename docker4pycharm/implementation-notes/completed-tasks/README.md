# Completed Task Archive

This folder stores retrospective notes for tasks that were removed from the
active handoff list.

Use this folder when a task is completed, manually validated, no longer
reproduced, or intentionally retired. The active task list in the root
`README.md` should stay short and should include only work the next agent should
actually consider doing.

Each completed task note should include:

- Date.
- Status.
- Original task.
- Done means.
- Verification or user validation.
- Environment provenance.
- Important notes for retrospective.
- Reopen conditions.

Do not put secrets in this folder. Record credential and external-state facts
only at the level needed for future debugging.
